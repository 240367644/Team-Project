# Accom4U (Team 49)
Accom4U is an e-commerce website designed to provide an easy way for students to search for and purchase essentials for their accommodations.

## Features
- Landing Page
- About Us
- Products Display
- Contact Form
- Functional Log In/Register
- Basket and Dummy Checkout
